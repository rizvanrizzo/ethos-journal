## Local Development
1. `npm install`
2. `npm run dev`

## AI Features
Powered by Google Gemini API.
- **Auto-Summarization**: Summarizes your journal entries into 2-3 lines.
- **Mood Detection**: Identifies the emotional tone and provides suggestions.
- **Writing Assistant**: Helps you continue your thoughts or improve your writing.

## Design
Premium, minimal, and emotionally resonant design inspired by high-end Flutter applications.
- Soft color palette (Sage/Cream).
- Fluid animations with `motion`.
- Responsive, mobile-first interface.
